# Label Africa AI

A modern, professional AI data labeling platform designed to connect African talent with global AI annotation opportunities. Built with React, TypeScript, and modern web technologies.

![Label Africa AI Dashboard](https://res.cloudinary.com/ecosheane/image/upload/v1754217265/lable_z2geih.png)

## 🌟 Features

### **Professional Dashboard**
- **Modern UI/UX**: Clean, professional interface inspired by leading AI platforms like Scale AI
- **Dark Mode Support**: Complete dark/light theme toggle for better user experience
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Real-time Metrics**: Live performance tracking and quality metrics

### **AI Annotation Platform**
- **Task Management**: Comprehensive task queue and assignment system
- **Quality Assurance**: Built-in quality scoring and performance tracking
- **Earnings Tracking**: Real-time earnings calculation and payment history
- **Project Management**: Organized project structure for different annotation types

### **Labeler Features**
- **Multi-format Support**: Image classification, audio transcription, text classification, video moderation
- **Performance Analytics**: Detailed metrics and progress tracking
- **Training Resources**: Access to training materials and skill development
- **Community Features**: Connect with other labelers and share experiences

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Modern web browser

### Installation

1. **Clone the repository:**
```bash
git clone <your-repo-url>
cd label-africa-ai
```

2. **Install dependencies:**
```bash
npm install
```

3. **Start the development server:**
```bash
npm run dev
```

The application will be available at `http://localhost:8080`
